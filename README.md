# ⏱️📋 Telegram Scheduler Bot
Телеграм-бот с планировщиком задач, регулярными прогнозами погоды и трекером рабочего времени.


## Возможности бота
• Отправка прогноз погоды по запросу;  
• Планирование разовое/регулярное сообщений;  
• Планирование разовое/регулярное действий из списка доступных действий бота;  
• рекер рабочего времени (в разработке).

## Запуск и настройка проекта.

• __Клонируйте репозиторий__ на хост бота:
```bash
git clone https://github.com/SgtPepeF/telegram_job_scheduler_bot.git
```
• В корне проекта __создайте виртуальную среду__ и установите зависимости: 
```bash
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```

• В корневой директории проекта __создайте файл ```.env```__, по примеру ```.env.example```. Внесите туда api-ключ своего ТГ-бота, api-ключ из профиля openweathermap [ https://openweathermap.org/ ]. Также обязательно заполните поля с регионом погоды, telegram-id и свой tg-username 

• Когда среда будет готова, можно запускать бот. Перед началом __выполните команду__ 
```bash
# создаст в корне проекта базу данных sqlite и добавит в неё все необходимые таблицы приложений бота.

python bot/db_creation.py
```

Опционально можете выполнить команду  
```bash
# Добавляет в БД регулярные задачи: прогноз погоды и базовое расписание дня.
python bot/populate_db.py
# В будущем вы можете модифицировать данный фвйл, добавив туда задачи по своему усмотрению.
```

• __Запуск бота__  осуществляется командой:
```bash
# Добавляет в БД регулярные задачи: прогноз погоды и базовое расписание дня.
python bot/main.py
```
Начните общение с ботом в чате телеграма с команды ```/start```. Бот отправит Вам дальнейшие инструкции по работе с ним.
